## 一、建模结果

本次实验选取的系统是OA系统，本系统开放给在在职员工申请审批。本次实验的建模结果包括结构化自然语言、UML类图、系统顺序图和用例图，具体如下图所示：
在职员工可使用本系统进行申请审批等操作，具体包括：登录系统、查看申请情况、创建申请、撤销申请、修改个人信息。系统管理员可以进行处理申请等操作，具体包括：登录系统、查看申请情况、处理申请、管理用户（添加、删除、修改）。
